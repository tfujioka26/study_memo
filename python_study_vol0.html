ディレクトリ内のファイルをリストアップ

1/3 2022

カテゴリー：基本操作

標準モジュールのglobを使います。
検索したい拡張子、フォルダなど指定できます。 

import glob
files = glob.glob("*.jpg")
print(files)

サブディレクトリまでリストアップするには、
"**"とrecursive=Trueを記入します。 

import glob
files = glob.glob("**/*.jpg", recursive=True)
print(files)
