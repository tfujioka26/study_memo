画像をイラスト風に変更

1/3 2022

カテゴリー：画像関連

opencvを使って、画像(写真)をイラスト風に変更できます。
色々な方法があるみたいですが、一例を紹介します。
全てのjpgをリストアップし、全て変換するコードです。 

import numpy as np
import cv2
import glob
files = glob.glob('*.jpg')

for i, file in enumerate(files):
    img = cv2.imread(file)
    dst = cv2.stylization(img, sigma_s=60, sigma_r=0.6)
    if i < 10:
        cv2.imwrite('illustration_0' + str(i) + '.png', dst)
    else:
        cv2.imwrite('illustration_' + str(i) + '.png', dst)

他、エッジを保持しつつ平滑化を行う(edgePreservingFilter)、
細部強調(detailEnhance)、鉛筆風(pencilSketch)、
油絵風（xphoto.oilPainting）等があります。
※ちなみに、xphotoは、opencv-pythonでは使えませんでした。
